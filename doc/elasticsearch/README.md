# Elasticsearch - 学习笔记
## 1. 简介
> `Elasticsearch`  是一个高度可伸缩的开源全文检索和分析引擎。它使你能够存储、查询、搜索、分析大量的数据，并且能够几乎实时的查询。通常被用实现应用程序复杂检索要求的底层技术引擎。

`Elasticsearch`可以用于以下场景：
- 在线购物商城；
- 海量日志分析平台；
- 商品价格预警；
- 商业智能分析；
